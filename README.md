# Fruitful Danger – 2D Platformer (Unity)


A 2D platformer game developed with Unity (2020.3 LTS). The project includes multiple levels, a main menu, checkpoint system, collectibles with scoring, enemies (melee + ranged), traps, health system, and UI flow (pause, game over, win).

### Features
- 3 playable levels + main menu
- Player movement: run + jump (with coyote time and extra jumps)
- Ranged attack: fireball shooting with cooldown
- Checkpoint / respawn system
- Collectibles (score) and health pickups
- Enemies: patrol, melee enemy, ranged enemy
- Traps: spikes, saw, spikehead, arrow trap, fire trap
- UI: pause (ESC), game over, win screen, volume controls
- Persistent scoring across levels (ScoreManager)

### Controls
- Move: A/D or Arrow Keys
- Jump: Space
- Attack: Left Mouse Button
- Pause: ESC

### How to Run
1. Open the project with **Unity 2020.3.38f1**
2. Open the menu scene: `Assets/Levels/_Menu.unity`
3. Press Play

### Project Structure
- `Assets/` – Scenes, prefabs, scripts, animations, audio
- `Packages/` – Unity package dependencies
- `ProjectSettings/` – Project configuration
